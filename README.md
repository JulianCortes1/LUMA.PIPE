## 🚀 Cómo publicar en GitHub Pages (paso a paso)

### Paso 1 — Crear repositorio en GitHub
1. Entra a [github.com](https://github.com) e inicia sesión
2. Haz clic en **"New repository"** (botón verde)
3. Ponle el nombre: `luma-english`
4. Déjalo en **Public**
5. Haz clic en **"Create repository"**

### Paso 2 — Subir los archivos
En la página del repositorio recién creado:
1. Haz clic en **"uploading an existing file"**
2. Arrastra o selecciona los 3 archivos:
   - `index.html`
   - `manifest.json`
   - `sw.js`
3. Haz clic en **"Commit changes"**

### Paso 3 — Activar GitHub Pages
1. Ve a **Settings** (pestaña del repositorio)
2. En el menú izquierdo, haz clic en **Pages**
3. En "Source" selecciona: **Deploy from a branch**
4. En "Branch" selecciona: **main** → carpeta **/ (root)**
5. Haz clic en **Save**

### Paso 4 — Obtener el link
Espera 1-2 minutos y GitHub te mostrará el link:
```
https://TU-USUARIO.github.io/luma-english/
```

¡Ese link se puede abrir en cualquier celular y instalar como app!

---

## 📱 Cómo instalar como app en el celular

**Android (Chrome):**
1. Abre el link en Chrome
2. Toca el menú (⋮) → "Añadir a pantalla de inicio"
3. Confirma — aparecerá como app en tu pantalla

**iPhone (Safari):**
1. Abre el link en Safari
2. Toca el botón compartir (□↑) → "Agregar a pantalla de inicio"
3. Confirma — aparecerá como app
